# Challenge-Alura-Store
Este proyecto forma parte del Challenge 1 de Data Science de Alura LATAM. El objetivo principal es realizar un análisis exhaustivo de cuatro sucursales de "AluraStore" para determinar cuál de ellas presenta el desempeño más débil y debería ser vendida por el Sr. Juan.

**1. Introducción**  
El programa procesa y analiza la información comercial de cuatro tiendas distintas cargadas desde fuentes externas en formato CSV. El análisis busca equilibrar factores cuantitativos (ingresos y costos) y cualitativos (satisfacción del cliente) para tomar una decisión de desinversión estratégica.
